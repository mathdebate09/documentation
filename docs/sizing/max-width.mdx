## Custom Max Widths

Custom properties are usually rendered using `_(customValue)`

> Max Width in React Native does not support `%` unit as of now

| Object             | Native Properties        |
| ------------------ | ------------------------ |
| w.max_(customValue)  | `{ maxWidth: customValue }` |
