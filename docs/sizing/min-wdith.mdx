Define the minimum width an element can have.

## Import

```js title="src/screens/Component.js"
import { w } from "nativeflowcss";
```

## Custom Min Widths

Custom properties are usually rendered using `_(customValue)`

> Min Width in React Native does not support `%` unit as of now

| Object             | Native Properties        |
| ------------------ | ------------------------ |
| w.min_(customValue)  | `{ minWidth: customValue }` |
