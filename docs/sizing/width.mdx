## Custom Widths

Custom properties are usually rendered using `_(customValue)`

> Width in React Native does not support `%` unit as of now

| Object             | Native Properties        |
| ------------------ | ------------------------ |
| w.w_(customValue)  | `{ width: customValue }` |
