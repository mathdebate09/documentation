## Custom Heights

Custom properties are usually rendered using `_(customValue)`

> Height in React Native does not support `%` unit as of now

| Object                       | Native Properties              |
| ---------------------------- | ------------------------------ |
| h.h_(customValue)  | `{ height: customValue }` |
