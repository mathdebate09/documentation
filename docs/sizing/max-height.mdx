## Custom Max Heights

Custom properties are usually rendered using `_(customValue)`

> Max Height in React Native does not support `%` unit as of now

| Object             | Native Properties        |
| ------------------ | ------------------------ |
| h.max_(customValue)  | `{ maxHeight: customValue }` |
