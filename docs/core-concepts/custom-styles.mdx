How to input your desired values for various utility style-objects.

## How to generate Custom Styles

- **Step 1**: Get the object you want to get custom generated and remove the value
- **Step 2**: Add `()` at the end of the the object ( using the function )
- **Step 3**: Give your desired input in the `string` format ( It also works with `number` so some properties )

### Example

We'll try using a custom background color, using `#ccc` instead of `slate_300`

- **Step 1**: `fx.bg_color_slate_300` âžœ `fx.bg_color_`
- **Step 2**: `fx. bg_color_()`
- **Step 3**: `fx.bg_color_("#ccc")`
