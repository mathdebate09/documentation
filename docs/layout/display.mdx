Set element display properties like hidden, flex.

## Import

```js title="src/screens/Component.js"
import { display } from "nativeflowcss";
```

## Properties

| Object        | Native properties         |
| ------------- | ------------------------- |
| display.hidden   | `{ display: 'none' }` |
| display.flex | `{ display: 'flex' }`      |
