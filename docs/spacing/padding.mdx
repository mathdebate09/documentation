Classes to add spacing inside element's border.

## Properties

| Classes       | Native Props                     |
|---------------|-------------------------------|
| `p.p(n)`        | `padding: n,`                 |
| `p.x(n)`       | `paddingLeft: n,` `paddingRight: n,` |
| `p.y(n)`       | `paddingTop: n,` `paddingBottom: n,` |
| `p.t(n)`       | `paddingTop: n,`             |
| `p.r(n)`       | `paddingRight: n,`           |
| `p.b(n)`       | `paddingBottom: n,`          |
| `p.l(n)`       | `paddingLeft: n,`            |
