Different classes used for constructing flexbox based layouts

## Properties

| Classes           | Native Props                     |
|-------------------|----------------------------------|
| `flex.f(n)`       | `flex: n,`                       |
| `flex.gap(n)`     | `gap: n,`                        |
| `flex.wrap(v)`    | `flexWrap: v,`                   |
| `flex.justify(v)` | `justifyContent: v,`             |
| `flex.items(v)`   | `alignItems: v,`                 |
| `flex.alignContent(v)` | `alignContent: v,`          |
| `flex.alignSelf(v)`    | `alignSelf: v,`             |
