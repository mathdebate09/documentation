Control how elements shrink when space is limited.

## Import

```js title="src/screens/Component.js"
import { flex } from "nativeflowcss";
```

## Properties

| Object        | Native properties   |
| ------------- | ------------------- |
| flex.shrink   | `{ flexShrink: 1 }` |
| flex.shrink_0 | `{ flexShrink: 0 }` |
