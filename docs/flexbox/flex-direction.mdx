## Properties

| Classes             | Native Props                          |
| ------------------- | ------------------------------------- |
| flex.row            | `{ flexDirection: 'row' }`            |
| flex.row_reverse    | `{ flexDirection: 'row-reverse' }`    |
| flex.col            | `{ flexDirection: 'column' }`         |
| flex.column_reverse | `{ flexDirection: 'column-reverse' }` |
